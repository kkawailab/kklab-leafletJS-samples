12 - ドラッグ可能なマーカー

学習内容: ユーザーがドラッグして位置を変更できるマーカーを作成します。

ポイント:

  • draggable: true オプションでドラッグを有効化
  • dragstart, drag, dragend イベントで位置変更を検知
  • getLatLng() で現在の座標を取得

操作: マーカーをドラッグして位置を変更してください。

マーカーの位置

緯度: 35.681200
経度: 139.767100 

// ドラッグ可能なマーカーを作成
const marker = L.marker([35.6812, 139.7671], {
    draggable: true
}).addTo(map);

// ドラッグ開始
marker.on('dragstart', function(e) {
    console.log('ドラッグ開始');
});

// ドラッグ中
marker.on('drag', function(e) {
    const pos = marker.getLatLng();
    console.log('移動中:', pos.lat, pos.lng);
});

// ドラッグ終了
marker.on('dragend', function(e) {
    const pos = marker.getLatLng();
    console.log('最終位置:', pos.lat, pos.lng);
});
