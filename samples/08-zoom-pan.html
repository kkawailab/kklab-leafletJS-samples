08 - 地図のズームとパン操作

学習内容: プログラムから地図のズームレベルや表示位置を制御します。

ポイント:

  • setView() で中心とズームを同時に設定
  • flyTo() でアニメーション付きの移動
  • fitBounds() で特定の範囲を表示
  • zoomIn() / zoomOut() でズーム操作
中心: -- , -- | ズーム: -- 
// 即座に移動
map.setView([35.6812, 139.7671], 15);

// アニメーション付きで飛ぶ
map.flyTo([34.6937, 135.5023], 13, {
    duration: 2  // 2秒かけてアニメーション
});

// 範囲にフィット
map.fitBounds([
    [31.0, 129.0],  // 南西
    [45.5, 145.8]   // 北東
]);

// ズーム操作
map.zoomIn();
map.zoomOut();

// パン（移動）
map.panBy([100, 0]);  // 100px右へ
