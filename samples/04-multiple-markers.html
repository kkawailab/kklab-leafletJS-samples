04 - 複数のマーカー

学習内容: 配列データから複数のマーカーを効率的に地図上に配置します。

ポイント:

  • 位置情報を配列で管理
  • forEach() でループしてマーカーを追加
  • 各マーカーに異なるポップアップを設定
// 駅データの配列
const stations = [
    { name: '東京駅', lat: 35.6812, lng: 139.7671 },
    { name: '新宿駅', lat: 35.6896, lng: 139.7006 },
    { name: '渋谷駅', lat: 35.6580, lng: 139.7016 },
    { name: '品川駅', lat: 35.6284, lng: 139.7387 },
    { name: '上野駅', lat: 35.7141, lng: 139.7774 }
];

// 配列をループしてマーカーを追加
stations.forEach(station => {
    L.marker([station.lat, station.lng])
        .addTo(map)
        .bindPopup(`<b>${station.name}</b>`);
});
