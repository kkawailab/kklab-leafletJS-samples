06 - ポリゴンを描画

学習内容: 地図上に多角形（ポリゴン）と矩形（四角形）を描画します。

ポイント:

  • L.polygon() は座標の配列から多角形を作成
  • L.rectangle() は対角の2点から矩形を作成
  • 穴あきポリゴンも作成可能（配列のネスト）
// 三角形のポリゴン
const triangle = L.polygon([
    [35.69, 139.70],
    [35.68, 139.72],
    [35.67, 139.70]
], {
    color: 'purple',
    fillOpacity: 0.4
}).addTo(map);

// 矩形
const rectangle = L.rectangle([
    [35.66, 139.74],
    [35.68, 139.76]
], {
    color: 'orange',
    weight: 2
}).addTo(map);
